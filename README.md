# Pratik Bhattarai - Portfolio

My personal portfolio website showcasing my projects, skills, and experience.

## Overview

This is my GitHub Pages portfolio site where I share information about myself, my work, and my development journey.

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling
- **GitHub Pages** - Hosting

## Features

- Personal portfolio showcase
- Project highlights
- About me section
- Responsive design
- Clean, professional layout

## Live Website

Visit my portfolio: [https://pratik-bhattarai12.github.io/](https://pratik-bhattarai12.github.io/)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/pratik-bhattarai12/pratik-bhattarai12.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd pratik-bhattarai12.github.io
   ```

3. Open `index.html` in your web browser or visit the live URL above
